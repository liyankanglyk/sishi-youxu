# 1. 后端
cd backend

mysql -u root -p < scripts/init_db.sql
mysql -u root -p sishi_youxu < scripts/seed.sql
python scripts/init_admin.py          # 创建默认超管 admin/123456
python -m uvicorn src.main:app --reload --port 8000

# 2. 验证后端
#    - 健康检查：         http://127.0.0.1:8000/health
#    - API 文档：         http://127.0.0.1:8000/docs

#    - POST /api/v1/users              注册新用户

#    - POST /api/v1/auth/wechat/login   微信登录（mock）

# 4. 验证管理端（Phase 1）
#    - POST /api/v1/admin/auth/tokens   admin/123456 管理员登录
#    - GET  /api/v1/admin/dashboard/stats  仪表盘统计
#    - GET  /api/v1/admin/users            用户列表
#    - GET  /api/v1/admin/audit            审计日志
#    - GET  /api/v1/admin/feedback         反馈管理
#    - GET  /api/v1/admin/config           系统配置

# 5. 验证管理端内容管理（新增）
#    - GET  /api/v1/admin/tasks            任务列表（管理员）

| 风险项 | 说明 | 决策点 |
|---|---|---|
| WeChat code2session 真实接入 | 需要 AppID/Secret；当前 `WX_LOGIN_MOCK=true` | 上线前必须切到真实模式 |
| 微信订阅消息模板 | 需在微信公众平台申请 `task_reminder` 模板 | 提前 1 周申请 |

| Alembic 迁移策略 | 当前用 SQLAlchemy `create_all`，生产不推荐 | Phase 2 前引入 |
