# 管理后台 UI 设计规范

> 版本：V1.0.0 | UI框架：Element Plus + ECharts | 更新日期：2026-06-27

---

## 整体布局

### 布局结构

```
┌──────────────────────────────────────────────────────────────────┐
│  顶部 Header（60px）                                              │
│  ┌────────┬────────────────────────────────────────────────┐    │
│  │ Logo   │ 面包屑              用户信息  通知  全屏          │    │
│  └────────┴────────────────────────────────────────────────┘    │
├────────────┬─────────────────────────────────────────────────────┤
│            │                                                     │
│  侧边栏    │              主内容区                                │
│  220px     │  ┌─────────────────────────────────────────────┐  │
│  (可折叠)  │  │  搜索区域（折叠/展开）                        │  │
│            │  ├─────────────────────────────────────────────┤  │
│  ┌──────┐ │  │  数据表格（斑马纹）                           │  │
│  │ Logo  │ │  ├─────────────────────────────────────────────┤  │
│  └──────┘ │  │  分页器（共 XX 条）                          │  │
│  一级菜单  │  └─────────────────────────────────────────────┘  │
│  ┌──────┐ │                                                     │
│  │ 菜单1 │ │                                                     │
│  └──────┘ │                                                     │
│  ┌──────┐ │                                                     │
│  │ 菜单2 │ │                                                     │
│  │ ┌────┐│ │                                                     │
│  │ │子菜单││ │                                                     │
│  │ └────┘│ │                                                     │
│  └──────┘ │                                                     │
└────────────┴─────────────────────────────────────────────────────┘
```

### 布局参数

Header 60px固定 | 侧边栏220px（折叠64px）| 主内容区 calc(100vh - 60px) | 表格行高48px | 分页器48px

背景色: #304156（深蓝灰）| 菜单文字: #BFCBD9 | 菜单高亮: #409EFF | 子菜单背景: #263445

`--el-color-success`: #67C23A | `--el-color-warning`: #E6A23C | `--el-color-danger`: #F56C6C | `--el-color-info`: #909399

页面背景: #F0F2F5 | 卡片背景: #FFFFFF | 表格斑马纹: #FAFBFC | 边框色: #E4E7ED

## 侧边栏导航

### 菜单结构

| 一级菜单 | 二级菜单 | 对应页面 |
|----------|----------|----------|
| 仪表盘 | — | `DashboardView` |
| 用户管理 | 用户列表 | `UserListView` |
|  | 用户详情 | `UserDetailView` |
|  | 登录日志 | `LoginLogView` |
| 任务管理 | 任务审核 | `TaskListView` |
|  | 标签审核 | `TagListView` |
| 内容管理 | 反馈管理 | `FeedbackView` |
| 系统管理 | 敏感词 | `SensitiveWordView` |
|  | IP 黑名单 | `IpBlacklistView` |
|  | 系统配置 | `ConfigView` |
|  | 公告 | `AnnouncementView` |
| 系统监控 | 操作日志 | `AuditView` |

### 菜单交互

默认: #BFCBD9无背景 | hover: #fff文字+#263445背景 | 激活: #409EFF文字+左边框3px主色 | 父级菜单: 图标+文字不可点击

### 侧边栏折叠

220px→64px，点击折叠按钮，折叠时只显示图标+tooltip，动画width 0.3s

## 数据表格

### 表格结构

操作列 | 编号 | 用户名 | 手机号 | 状态 | 创建时间 | 操作（编辑/删除）

### 表格特性

斑马纹 #FAFBFC/#FFFFFF | hover #ECF5FF | 多选checkbox | 列头排序 | 列宽自适应，最小100px

### 分页器

左侧"共 {total} 条" | 页码居中（<7页显示全部，>7页显示1...N）| 每页条数下拉: 10/20/50/100 | 跳转输入框+按钮

---

## 表单设计

### 表单布局

label 100px宽右对齐#606266 | 输入框宽300px高34px | textarea宽500px高80px | 必填红色*前缀

### 搜索表单

默认折叠只显示关键条件，点击"展开"显示更多，搜索区卡片白背景+阴影

---

## 按钮设计

| 类型 | 样式 | 用途 |
|------|------|------|
| 主按钮 | #1890FF白字 | 提交确认 |
| 默认按钮 | #FFFFFF#DCDFE6边框 | 取消返回 |
| 文字按钮 | 无边框背景 | 重置更多 |
| 危险按钮 | #F56C6C白字 | 删除 |
| 成功按钮 | #67C23A白字 | 启用解封 |

尺寸：大36px/中32px/小24px

按钮组合：左侧主要操作[新增][导出][批量删除] | 右侧行操作[编辑][删除][更多▼]

左侧: 折叠按钮(汉堡图标) + 面包屑 | 右侧: 全屏图标 + 通知(红点) + 圆形32px头像 + 下拉菜单(个人中心/登出)
