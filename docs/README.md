## 项目概述

**四时有序** 是一款基于艾森豪威尔矩阵的个人任务管理工具。

- Vue 3 + FastAPI 全栈
- 多用户 + 云端同步（离线优先）
- 管理后台（Element Plus + ECharts）
- 支持 PWA / 移动端扩展（后续阶段）

### 技术栈

| 类别 | 技术 |
|------|------|
| 后端 | Python 3.12 + FastAPI + SQLAlchemy 2.0 (async) + MySQL + Redis |
| 前端 | Vue 3 + TypeScript + Vite + Pinia + Dexie.js |
| 管理后台 | Vue 3 + Element Plus + ECharts |
| 认证 | JWT + 多 Provider（password/phone_sms/email_code/wechat）|
| 同步 | 离线优先 + 冲突 last-write-wins |

### 环境要求

Python 3.12+ | Node.js 20+ | MySQL 8.0+ | Redis 6.0+ | Nginx 1.18+

| Phase 0 | 项目骨架 | 1 周 | ✅ 已完成 | 目录结构、框架、数据库、API 规范 |
| Phase 1 | 管理端核心 | 4-5 周 | ✅ 已完成 | 后端 API + 管理端前端完整实现 |
| Phase 2 | 用户端核心 | 2 周 | ✅ 已完成 | 用户端任务/标签/通知/反馈/同步后端 API |
| Phase 3 | 认证与同步 | 2 周 | ⚠️ 后端完成 / 前端 0% | 多 Provider 登录、微信 code2session、ws-ticket；用户端前端待开发 |

| Phase 5 | 用户端增强 | 2-3 周 | ❌ 未开始 | 四象限拖拽、撤销重做、视图密度、快捷键 |
| Phase 6 | 测试与部署 | 1-2 周 | ⚠️ 仅烟雾 | 单元 / E2E / CI/CD、Alembic、APScheduler、Sentry |
