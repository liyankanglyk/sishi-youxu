## 表结构

### 用户表 (sishiyouxu_user)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| nickname | VARCHAR(50) | 昵称 | NOT NULL |
| avatar_url | VARCHAR(500) | 头像 URL | |
| role | ENUM('user','admin','super_admin') | 角色 | DEFAULT 'user' |
| status | ENUM('active','disabled','banned','deleted') | 状态 | DEFAULT 'active' |
| locale | VARCHAR(10) | 语言偏好 | DEFAULT 'zh-CN' |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_role_status`, `idx_created_at`, `idx_deleted_at`

### 认证身份表 (sishiyouxu_auth_identity)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 关联用户 | → sishiyouxu_user.uuid |
| provider | ENUM('password','phone_sms','email_code','wechat') | 登录方式 | NOT NULL |
| provider_uid | VARCHAR(255) | 第三方 UID（微信登录存 openid）| NOT NULL |
| credentials | TEXT | 加密凭证（密码哈希等）| |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`uk_provider_uid` (provider + provider_uid 联合唯一)，`idx_user_uuid`, `idx_deleted_at`

### Refresh Token 表 (sishiyouxu_refresh_token)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 关联用户 | → sishiyouxu_user.uuid |
| jti | VARCHAR(64) | Token 唯一标识 | UNIQUE |
| token_hash | VARCHAR(64) | Token 哈希（SHA-256）| NOT NULL |
| expires_at | DATETIME | 过期时间 | NOT NULL |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| revoked_at | DATETIME | 吊销时间 | |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_user_uuid`, `idx_jti`, `idx_expires_at`, `idx_deleted_at`

> **撤销语义**：`revoked_at` 表示主动撤销（登出/踢下线），`deleted_at` 表示过期清理；两条语义并存。

### 任务表 (sishiyouxu_task)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 所属用户 | → sishiyouxu_user.uuid, NOT NULL |
| title | VARCHAR(255) | 任务标题 | NOT NULL |
| urgency_level | INT | 紧急度等级（-4..4，0 为分界线）| NOT NULL DEFAULT 0 |
| importance_level | INT | 重要度等级（-4..4，0 为分界线）| NOT NULL DEFAULT 0 |
| due_date | DATE | 截止日期 | |
| recurrence | TEXT | 重复规则，RRULE 格式（含 EXDATE/RDATE） | |
| note | TEXT | Markdown 备注 | |
| completed | BOOLEAN | 完成状态 | DEFAULT FALSE |
| completed_at | DATETIME | 完成时间 | |
| sort_order | INT | 排序序号 | DEFAULT 0 |
| remind_at | DATETIME | 首次提醒时间 | |
| remind_offset_minutes | INT | 提前提醒分钟数（NULL=使用 due_date 当天 9:00） | |
| reminder_state | ENUM('pending','sent','cancelled') | 提醒状态 | DEFAULT 'pending' |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_user_completed` (user_uuid, completed)，`idx_user_due_date` (user_uuid, due_date)，`idx_user_remind` (remind_at, reminder_state)，`idx_updated_at`，`idx_deleted_at`

> **循环任务边界**：`recurrence` 字段支持完整 RFC 5545 RRULE，可包含 EXDATE/RDATE 等长字符串；超出 `VARCHAR(100)` 时使用 `TEXT`。
> **提醒机制**：`remind_at` 为任务应被提醒的绝对时间点；后端调度器扫描 `idx_user_remind` 触发推送，触发后 `reminder_state` 置为 `sent`；循环任务下一次触发时间由后端在 `sent` 后重新计算并写回。

### 标签表 (sishiyouxu_tag)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 所属用户（预设标签为 NULL）| → sishiyouxu_user.uuid |
| name | VARCHAR(50) | 标签名称 | NOT NULL |
| color | VARCHAR(9) | 颜色 HEX | NOT NULL |
| is_preset | BOOLEAN | 是否预设 | DEFAULT FALSE |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_user_uuid`, `idx_user_name` (user_uuid, name)，`idx_deleted_at`

> **唯一性**：`idx_user_name` 为普通索引，同名标签唯一性由应用层保证（MySQL 不支持带 `WHERE deleted_at IS NULL` 的部分唯一索引）。

### 任务-标签关联表 (sishiyouxu_task_tag)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| task_uuid | CHAR(36) | 关联任务 | → sishiyouxu_task.uuid |
| tag_uuid | CHAR(36) | 关联标签 | → sishiyouxu_tag.uuid |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**主键**：`PRIMARY KEY (task_uuid, tag_uuid)`
**索引**：`idx_tag_uuid` (tag_uuid)，`idx_deleted_at`

### 检查项表 (sishiyouxu_task_checklist)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| task_uuid | CHAR(36) | 所属任务 | → sishiyouxu_task.uuid, NOT NULL |
| title | VARCHAR(255) | 检查项标题 | NOT NULL |
| completed | BOOLEAN | 完成状态 | DEFAULT FALSE |
| sort_order | INT | 排序序号 | DEFAULT 0 |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_task_sort` (task_uuid, sort_order)，`idx_deleted_at`

### 审计日志表 (sishiyouxu_audit_log)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 操作用户 | → sishiyouxu_user.uuid |
| action | VARCHAR(100) | 操作类型 | NOT NULL |
| resource_type | VARCHAR(50) | 资源类型 | NOT NULL |
| resource_uuid | CHAR(36) | 资源 UUID | |
| ip_address | VARCHAR(45) | IP 地址 | |
| user_agent | VARCHAR(500) | User Agent | |
| detail | JSON | 详细数据 | |
| created_at | DATETIME | 操作时间 | NOT NULL |

**索引**：`idx_user_action` (user_uuid, action)，`idx_resource` (resource_type, resource_uuid)，`idx_created_at`

> **append-only**：审计日志不支持修改/软删除；超过保留期（建议 180 天）由后台归档任务迁移到冷存储。

### 公告表 (sishiyouxu_announcement)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| title | VARCHAR(200) | 公告标题 | NOT NULL |
| content | TEXT | 公告内容 | NOT NULL |
| type | ENUM('info','warning','critical') | 公告类型 | DEFAULT 'info' |
| is_pinned | BOOLEAN | 是否置顶 | DEFAULT FALSE |
| is_active | BOOLEAN | 是否生效 | DEFAULT TRUE |
| start_time | DATETIME | 开始时间 | |
| end_time | DATETIME | 结束时间 | |
| created_by | CHAR(36) | 创建人 | → sishiyouxu_user.uuid |
| created_at | DATETIME | 创建时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_active_time` (is_active, start_time, end_time)，`idx_deleted_at`

> **生效判定**：`is_active = TRUE AND (start_time IS NULL OR start_time <= NOW()) AND (end_time IS NULL OR end_time >= NOW())`。前端拉取时直接由该索引覆盖。

### 通知表 (sishiyouxu_notification)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 通知所属用户 | → sishiyouxu_user.uuid, NOT NULL |
| kind | VARCHAR(32) | 通知类型 | NOT NULL |
| is_read | BOOLEAN | 是否已读 | DEFAULT FALSE |
| read_at | DATETIME | 已读时间 | |
| title | VARCHAR(200) | 通知标题 | NOT NULL |
| body | TEXT | 通知内容 | NOT NULL |
| task_uuid | CHAR(36) | 关联任务 UUID | |

### 用户反馈表 (sishiyouxu_feedback)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| uuid | CHAR(36) | 主键 | PK |
| user_uuid | CHAR(36) | 反馈用户 | → sishiyouxu_user.uuid |
| content | TEXT | 反馈内容 | NOT NULL |
| contact | VARCHAR(100) | 联系方式（可选）| |
| status | ENUM('pending','processing','resolved') | 处理状态 | DEFAULT 'pending' |
| handled_by | CHAR(36) | 处理人 | → sishiyouxu_user.uuid |
| handled_at | DATETIME | 处理时间 | |
| created_at | DATETIME | 反馈时间 | NOT NULL |
| updated_at | DATETIME | 更新时间 | NOT NULL |
| deleted_at | DATETIME | 软删除时间 | |

**索引**：`idx_user_uuid`, `idx_status`, `idx_created_at`, `idx_deleted_at`

> **状态机**：`pending`（用户提交）→ 管理员首次处理 `processing` → 处理完成 `resolved`。用户端只读 `pending` / `resolved`。

### 管理员权限表 (sishiyouxu_admin_permission)

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| role | ENUM('super_admin','admin') | 角色 | PK |
| permission | VARCHAR(100) | 权限节点 | PK |
| created_at | DATETIME | 创建时间 | NOT NULL |

**主键**：`PRIMARY KEY (role, permission)`

**预设权限节点**：

| 权限节点 | 说明 | super_admin | admin |
|----------|------|-------------|-------|
| `user:read` | 查看用户列表和详情 | ✅ | ✅ |
| `user:update` | 更新用户信息/状态 | ✅ | ✅ |
| `user:delete` | 删除用户 | ✅ | ❌ |
| `user:force_logout` | 强制登出用户 | ✅ | ✅ |
| `user:export` | 导出用户数据 | ✅ | ❌ |
| `task:audit` | 内容审核 | ✅ | ✅ |
| `task:delete` | 批量删除违规任务 | ✅ | ✅ |
| `sensitive_word:manage` | 敏感词库管理 | ✅ | ❌ |
| `sensitive_word:toggle` | 敏感词过滤开关 | ✅ | ❌ |
| `system_config:read` | 查看系统配置 | ✅ | ✅ |
| `system_config:write` | 修改系统配置 | ✅ | ❌ |
| `backup:create` | 创建数据备份 | ✅ | ❌ |
| `backup:restore` | 恢复数据备份 | ✅ | ❌ |
| `announcement:manage` | 公告管理 | ✅ | ✅ |
| `preset_tag:manage` | 预设标签管理 | ✅ | ✅ |
| `ip_blacklist:manage` | IP 黑名单管理 | ✅ | ❌ |
| `audit_log:read` | 审计日志查看 | ✅ | ✅ |
| `admin:manage` | 管理员账号管理 | ✅ | ❌ |
